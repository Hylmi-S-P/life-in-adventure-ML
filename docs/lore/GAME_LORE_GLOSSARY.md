# Game Lore Glossary

> Key terms, locations, factions, and characters from *Life in Adventure* (Studio Wheel). This glossary serves as a quick-reference for context when AI recommendations reference game-specific entities.

---

## 1. Factions & Guilds

| Term | Type | Description |
|------|------|-------------|
| Adventurer's Guild | Faction | Central hub for quests; players register here at game start |
| Temple of Light | Faction | Religious order; associated with Holy/Good epilogue paths |
| Dark Brotherhood | Faction | Underground syndicate; associated with Evil/Impure alignment paths |
| Merchant's Union | Faction | Trade organization; affects gold economy and shop access |
| Royal Guard | Faction | Kingdom's military; appears in main quest "Black Guardian" |

## 5. Backgrounds (DLC / Starting Stories)

Each background modifies available events, quest text, and exclusive epilogues:

| Background | Starting Stats | Unique Feature |
|------------|---------------|----------------|
| **Knight** | STR +2, CON +1 | Royal Guard questline access |
| **Mage** | INT +2, WIS +1 | Extra spellcasting events |
| **Rogue** | DEX +2, CHA +1 | Thief guild questline; extra gold |
| **Cleric** | WIS +2, CON +1 | Temple of Light events; healing bonus |
| **Noble** | CHA +2, gold +500 | Merchant discount; unique social events |
| **Commoner** | Balanced | Default experience; most flexible |

## 6. Epilogue Types

Endings unlocked based on alignment + EXP threshold at death:

| Epilogue | Requirement | Description |
|----------|-------------|-------------|
| "Remembered" | Good + EXP < 35 | Died early but remembered fondly |
| "Friendly Neighbourhood Adventurer" | Good + 35-80 EXP | Well-liked local hero |
| "Heroic Memories" | Good + EXP ≥ 80 | Legendary hero status |
| "Famous Adventurer" | Neutral + EXP ≥ 80 | Renowned but morally ambiguous |
| "Celebrated" | Evil + EXP ≥ 80 | Infamous but respected |
| Canon Ending | High INT + Good alignment | Best ending (verified via Fandom) |
